# Projetos em Java

Este diretório contém **projetos em Java** organizados dentro de subpastas.

## 📌 Sobre os projetos

- Os projetos presentes aqui são **amadores**.
- Foram desenvolvidos com o objetivo de **aprendizado e testes** da linguagem **Java**.
- Não representam aplicações finais, profissionais ou prontas para produção.
- Cada pasta pode conter exercícios, exemplos de código, testes de conceitos ou pequenos programas.

## 🎯 Objetivo

O principal objetivo deste diretório é:

- Praticar conceitos fundamentais da linguagem Java
- Compreender melhor orientação a objetos
- Testar estruturas, classes, métodos e APIs
- Registrar a evolução no aprendizado da linguagem
